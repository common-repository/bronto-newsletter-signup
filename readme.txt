Display a bronto signup form anywhere on your wordpress website with a simple shortcode.

== Description ==

This plugin will allow you to display a bronto signup form by using the shortcode [bronto_signup] anywhere on your wordpress website.


== Installation ==

Wordpress:
1. Search for Bronto Newsletter Signup
2. Click Install
3. Press Activate

Uploading:
1. Download sbd-bronto-newsletter.zip
2. Upload `sbd-bronto-newsletter` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
